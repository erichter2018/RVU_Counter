WHAT HAPPENS FOR THE USER
================================================================================

FOR NEW USER (Fresh Install):
1. User creates folder (e.g., Desktop\RVU_Counter)
2. User copies .bat file to that folder
3. User double-clicks .bat file
4. Script downloads latest version from GitHub
5. Script creates data/, settings/, logs/, helpers/ folders
6. Script creates empty database
7. Script launches RVU Counter
8. User is ready to start tracking!

FOR EXISTING USER (Upgrade):
1. User goes to their RVU Counter folder
2. User copies .bat file to that folder
3. User double-clicks .bat file
4. Script detects existing installation
5. Script backs up database and settings
6. Script downloads latest version
7. Script preserves all data
8. Script launches upgraded RVU Counter
9. What's New window appears
10. User continues with all their data intact!

Time required: 2-3 minutes per user

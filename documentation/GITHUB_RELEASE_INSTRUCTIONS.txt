# RVU Counter v1.7 - Major Update

## üöÄ Major Features

### Auto-Update System
- **One-Click Updates**: Check for new versions automatically on startup
- **No More Manual Downloads**: Updates download and install seamlessly
- **Update Notification**: Small "‚ú® Update Available" button appears when new version is ready
- **Safe Rollback**: Previous version kept as backup during update

### Integrated Tools (New!)
- **Database Repair Tool**: Scan and fix RVU mismatches directly from the app
- **Excel Payroll Checker**: Compare Excel files with RVU rules to find discrepancies
- **Easy Access**: Click the wrench icon (üîß) in the top right corner

### What's New Viewer (New!)
- **In-App Release Notes**: See what's new directly in the application
- **Access**: Click the "?" button in the Settings window

### Improved Data Organization
- **Separated Settings**: User preferences now separate from RVU rules
- **Cleaner Folder Structure**: 
  - `data/` - Your database and records
  - `settings/` - Your personal preferences
  - `helpers/` - Update tools
  - `logs/` - Application logs
- **Faster Rule Updates**: RVU classification rules can now be updated without app restart

## üîß Bug Fixes & Improvements

### RVU Classification Enhancements (19 fixes)
- ‚úÖ Fixed: "CT Abdomen" now correctly classified as 1.0 RVU (not CT AP)
- ‚úÖ Fixed: "MR Brain Angiography" now correctly gets 2.3 RVU
- ‚úÖ Fixed: Bilateral XR studies (AC joints, calcaneus, etc.) now get correct 0.6 RVU
- ‚úÖ Fixed: "XR Abdomen Decubitus" no longer matches ultrasound
- ‚úÖ New: "CT Outside Film Read" - 0.9 RVU
- ‚úÖ New: "XR Scanogram" - 1.0 RVU
- ‚úÖ New: "MRI Hip Bilateral" - 3.5 RVU
- ‚úÖ New: "CT Triple Spine" (C+T+L) - 5.25 RVU
- ‚úÖ New: "CT Head Face and Cervical" - 2.9 RVU
- ‚úÖ New: "CTA Lower Extremity" - 1.75 RVU
- ‚úÖ Updated: "CTA Chest + CT AP" - 3.0 RVU (was 2.68)
- ‚úÖ Updated: "CT Extremity Bilateral" - 1.0 RVU (was 2.0)

### Performance & Stability
- Improved startup speed with new settings architecture
- Better error handling for file operations
- Enhanced logging for troubleshooting

## üìã Installation

### For New Users:
1. Download `Install_or_Upgrade_RVU_Counter.bat`
2. Create a folder for RVU Counter
3. Run the batch file - it will download and set everything up

### For Existing Users (v1.6 or earlier):
1. Download `Install_or_Upgrade_RVU_Counter.bat`
2. Place it in your RVU Counter folder
3. Run the batch file - your data will be preserved

### Manual Installation:
1. Download `RVU Counter.exe`
2. Place in a folder of your choice
3. Run it - folder structure will be created automatically

## ‚ö†Ô∏è First Launch (v1.7)
When you first run v1.7, the app will automatically:
1. Create the new folder structure (`data/`, `settings/`, `helpers/`, `logs/`)
2. Move your database to `data/rvu_records.db`
3. Split your settings into user preferences and RVU rules
4. Keep a backup of your old `rvu_settings.yaml` file

**This is a one-time process and happens automatically. No action needed!**

================================================================================
USER INSTRUCTIONS (SIMPLE)
================================================================================

For Fresh Install:
1. Create new folder (e.g., Desktop\RVU_Counter)
2. Copy Install_or_Upgrade_RVU_Counter.bat to that folder
3. Double-click it
4. Follow prompts
5. Done!

For Upgrade:
1. Go to existing RVU Counter folder
2. Copy Install_or_Upgrade_RVU_Counter.bat to that folder
3. Double-click it
4. Follow prompts
5. Done! (All data preserved)

================================================================================

1. FRESH INSTALL:

   - Verify download works
   - Verify app launches
   - Verify empty database created

2. UPGRADE:
   - Existing v1.6 installation
   - Run script
   - Verify backup created
   - Verify data preserved
   - Verify app launches with data intact

QUICK COMMANDS
================================================================================

Create distribution package:
  powershell -ExecutionPolicy Bypass -File Create_Distribution_Package.ps1

Or manually create:
  New-Item -ItemType Directory -Path Distribution -Force
  Copy-Item Install_or_Upgrade_RVU_Counter.bat Distribution\
  Copy-Item DISTRIBUTION_README.txt Distribution\
  Compress-Archive Distribution\* -DestinationPath RVU_Counter_Install.zip

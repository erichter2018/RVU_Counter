# RVU Counter Releases

This repository hosts releases for the RVU Counter application.

## Download Latest Version

Go to [Releases](https://github.com/erichter2018/RVU-Releases/releases/latest) to download the latest version.

## Installation

1. Download `RVU Counter.exe` from the latest release
2. Run the executable - it will create the necessary folder structure automatically

For more information, see the release notes.
